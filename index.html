All The Places

A growing set of web scrapers designed to output consistent geodata about as many places of business in the world as possible.

Format

The linked file is a gzipped ndjson where each line is a GeoJSON feature.

Information about the properties contained in each GeoJSON feature are documented on the project’s GitHub repository.
